Descripción detallada

Clase generica y abstracta cuyo propósito es asegurarse que solo exista una instancia de una clase y proporciona un acceso fácil y global a la clase.

Parámetros del template
TClase que hereda a Singleton para que solo pueda haber una instancia y el acceso global de ella.
Restriciones de tipo
T :Component 

Documentación de las funciones miembro

void Singleton< T >.Awake ( )
protected

Al despertar el objeto al que este unido una clase en ejecucion que herede de Singleton, comprobara si ya hay una instancia previa de dicha clase.

En caso de que no haya ninguna instancia previa, asigna instance a la clase en ejecucion actual, en caso contrario, significa que ya habia una instancia, y destruye esta instancia de esta clase en ejecucion.

abstract void Singleton< T >.InitializeOnAwake ( )
protectedpure virtual

Metodo para inicializar otros atributos de la clase heredera de Singleton durante el metodo Awake

Implementado en PuzzleMapManager, ModalWindowHandler, NavigationManager, MultipleChoiceManager, GameFileManager y SettingsFileManager.

void Singleton< T >.OnApplicationQuit ( )
protected

En caso de cerrar el juego, el atributo instance se desasigna para que el recolector de basura elimine los datos de memoria.

void Singleton< T >.OnDestroy ( )
protected

En caso de destruirse el objeto al que esta unido una clase Singleton, si el atributo instance apuntaba a esta clase en ejecucion en concreto, la desasigna para indicar que actualmente no hay ningun Singleton de esta clase en concreto.
