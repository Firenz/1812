Descripción detallada

Clase generica y abstracta cuyo propósito es asegurarse que solo exista una instancia de una clase, proporcionar un acceso fácil y global a la clase, y que el objeto al que este unido una clase heredera de esta no se borre al cambiar de escena.

Documentación de las funciones miembro

void PersistentSingleton< T >.Awake ( )
protected

Al despertar el objeto al que este unido una clase en ejecucion que herede de PersistentSingleton, comprobara si ya hay una instancia previa de dicha clase.

En caso de que no haya ninguna instancia previa, asigna instance a la clase en ejecucion actual y llama al metodo DontDestroyOnLoad para que el objeto al que esta unido la clase de la instancia no se borre. En caso de que exista una instancia previa, significa que ya habia una instancia, y destruye esta instancia de esta clase en ejecucion.
