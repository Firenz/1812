Descripción detallada

Clase abstracta para boton generico de las interfaces graficas de los menus presentes en el juego.

protectedpure virtual

Metodo abstracto que debe ser sobrecargado en las clases herederas para definir las acciones a realizar por el boton al ser pulsado.

virtual void UIGenericButton.ChangeButtonTextLanguage ( )
protectedvirtual

Metodo que se ejecuta en caso de que suceda el evento de que se cambie el idioma, en ese caso, comprueba si el boton tiene el componente Text, si es asi, entonces le cambia el texto localizado por el del nuevo idioma actual.

virtual void UIGenericButton.OnMouseExit ( )
virtual

Raises the mouse exit event.

Metodo que se ejecuta cuando se inicia el evento de que el cursor ha dejado de estar encima del boton. Comprueba que el boton es interactivo y que el cursor no esta oculto, si es asi, cambia el grafico del cursor al que hay por defecto.

virtual void UIGenericButton.OnMouseOver ( )
virtual

Metodo que se ejecuta cuando se inicia el evento del cursor encima del boton.

Comprueba que el boton es interactivo y que el cursor no esta oculto, si es asi, cambia el grafico del cursor al de estar encima de un elemento de la interfaz grafica con CustomCursorController.ChangeCursorOverUIElement.

CustomCursorController

Reimplementado en LocalizationMarkButton.

virtual void UIGenericButton.Start ( )
protectedvirtual

Al iniciarse la instancia, obtiene el componente Button del objeto al que esta unido esta clase.

Despues intenta obtener el componente Text del boton del objeto y le asigna el texto localizado, en caso de que no posea el componente Text, se inicializa a null.

Reimplementado en GameSlotButton, ReturnToButton, MenuButton, ReturnToMainMenuButton, GadipediaSearchResultElementButton, JournalEntryDescriptionButton, CurrentResolutionButton, LocalizationMarkButton, InGameMenuButton, ChangeResolutionButton, JournalEntryButton y ChangeCurrentLanguageButton.

IEnumerator UIGenericButton.WaitForSFXAndExecuteClickAction ( )
protected

Corrutina que espera que se ejecute el sonido de pulsado del boton, y luego llama al metodo UIGenericButton.ActionOnClick que contiene las acciones a realizar por el boton al ser pulsado.
